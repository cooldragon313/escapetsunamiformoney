# Changelog

## Unreleased

### 新
- **重置進度按鈕**：角色創建畫面底部紅色虛線「⚠ 重置所有進度」連結，按下後彈出第一個紅色確認窗 → 再彈第二個深紅色「最後確認」窗 → 才會清掉 localStorage 並重新載入。雙保險防誤觸。

### 修
- 客戶端會偵測 server 是否為 Phase 1B（看 init 訊息有沒有 coins / waves 陣列）。如果遇到舊版 server（只認位置中繼），自動切回本地金幣 / 海嘯生成，避免「金幣和海嘯都不見」的空畫面狀況。

### 新
- **BGM 上線**：`assets/bgm.mp3` 會自動 loop 當背景音樂，按「開始冒險」時啟動（瀏覽器要求需用戶互動）。如果檔案還在下載就先記下來等載完再播。HUD 的 🔊 按鈕同時控制 BGM。

### 新（音效系統）
- **15 種音效合成**：跳、落地、撿金幣（叮叮）、撿死亡寶箱（鏘鏘鏘）、進坑、爬出、跳高不夠（拒絕嗶嗶）、海嘯警報、被打死、重生（升調叮咚）、存款（一堆銅板嘩啦）、購買升級（叮咚勝利）、暴風雨來臨（轟隆）、暴風雨結束、首次到山頂
- 4 個重點事件（警報 / 死亡 / 存款 / 暴風雨）支援檔案覆蓋：把 mp3 放 `assets/sfx/` 對應檔名即可
- BGM hook：放 `assets/bgm.mp3` 會自動 loop 播放
- HUD 加「🔊 / 🔇」靜音按鈕，偏好存 localStorage
- 第一次點「開始冒險」解鎖 audio context（瀏覽器要求需用戶互動）
- 所有音量混好不刺耳

### 新（Phase 1B 共享世界）
- **同一條海嘯**：海嘯生成移到 server，server 為權威，所有玩家看到同一條浪在同位置以同速度推進
- **金幣搶先撿**：玩家走到金幣 → 送 `pickup_attempt` 給 server → server 確認誰先到 → 廣播 `coin_pickup`，唯一贏家加進背包
- **共享暴風雨**：暴風雨啟動 / 結束由 server 廣播，所有人天空同時轉紅 / 復原
- **金幣 60 秒重生**：server 端計時，到時廣播 `coin_respawn`，所有人同時看到金幣重新出現
- 撿取改為等 server 確認才入袋，避免雙方都以為自己撿到
- 新加入的玩家會收到完整快照（玩家 / 可用金幣 / 目前進行中海嘯 / 暴風雨狀態）
- 海嘯位置用 `spawnTime + speed * elapsed` 計算，所有客戶端推算結果相同（確保同步）
- 死亡寶箱仍是各自的（暫時，下版可考慮共享）

### 新（Phase 1A 多人連線）
- **多人連線正式啟用**：`PARTYKIT_HOST` 設為 `etfm.cooldragon313.partykit.dev`
- **PartyKit server**（`party/server.js`）做位置中繼，所有玩家共用一個 `global` 房間
- 看得到其他玩家：客戶端建出他人的角色模型（套他們選的造型），頭上有名牌
- 進場 / 離場 toast，內插平滑移動，斷線後 5 秒自動重連
- `PARTYKIT_HOST` 留空 = 單人模式；填入 deploy 後的網址 = 多人模式
- 同步資料：位置、Y 軸轉向、移動 / 蹲坑 flag、名字、整套造型

### 新
- **開場故事畫面** + 「ESCAPE TSUNAMI / FOR MONEY」標題，鋪陳遠處寶藏的傳說
- **角色創建**：名字 / 髮型（短/帽/尖刺）/ 髮色 / 表情（笑/嚴肅）/ 膚色 / 上衣 / 褲子 / 鞋子；選擇透過 localStorage 儲存
- 玩家 3D 模型隨造型重建（眼睛 / 嘴巴 / 髮型 / 鞋子）
- **新手教學**：6 步驟講解操作、海嘯類型、坑跳高需求、存款流程、死亡掉寶；看過後不再重複
- 遊戲邏輯在選單期間暫停，3D 場景照常渲染
- **金幣每 60 秒自動補充**：被撿過的位置重新出現一個金幣（不影響死亡寶箱），確保長時間玩有持續收入升級

### 改
- 撿死亡寶箱不再直接入帳，改成佔 1 格背包、要帶回山頂才存款

### 修
- **坑爬出來瞬間又被吸回去**：3 秒避難結束後玩家還站在坑橢圓內 → 自動偵測再進坑。已紀錄「剛離開的坑」，走出範圍才能再進入
- **進度沒存**：先前每次重新打開都從 0 開始（金錢、升級、最遠距離）。現在用 localStorage 保留：存款 / 5 種升級等級 / 最遠距離；存款入帳和買升級時即時存，關分頁也會在 `beforeunload` 寫回
- **平板搖桿不出現**：原本只在初次載入偵測，部分平板 UA 沒回報觸控就跳過。改成「第一次摸畫面就啟動」，外加 `T` 鍵手動切換
- **手機搖桿卡住，角色一直前進**：手指拖出搖桿圓圈後 `pointerup` 在別的元素觸發，stick 元素上的 handler 收不到 → 卡在最後位置。move / up / cancel 改綁 `window`，加 `touchend` 備援當所有手指離開時強制歸零

## 2026-05-06

### 修
- **嚴重 bug**：海嘯被建立、計算碰撞、卻沒被加進 3D 場景 → 玩家看不到浪卻照樣被打死。修了。

### 改
- **坑系統大改**：從「時間陷阱」改成「跳高等級門檻」。跳高不夠的坑走過去顯示提示，不會把你關住。進得去的坑固定避難 3 秒。
- 坑緣顏色依區域標示需要的跳高 Lv（棕 / 綠 / 藍 / 紫 / 紅）
- 第 1 個坑從 −22m 推到 −50m，給開場 50m 暖身空間
- 第 1 區坑視覺縮小成 65%（給新手）

### 新
- **虛擬搖桿** + 觸控按鈕（跳 / 衝刺 / 商店），手機平板自動偵測切換
- viewport meta 防止手機雙指縮放
- 跑道**寬度 20 → 40 人身**
- **6 種海嘯類型**（顏色 × 高度 × 寬度 × 速度）：綠矮 / 藍標準 / 紅快 / 寬 / 紫巨 / 黑災難
- 部分寬度海嘯會**左右擺動**，可以朝反方向閃
- **隨機海嘯節奏**：60% 一般、25% 短間隔、15% 長平靜
- **暴風雨**：偶發 14–22 秒，海浪密集、寶藏 ×2、天空轉紅、罕見浪變多
- 邊景（草地、山脈）配合更寬跑道往外推
