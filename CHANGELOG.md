## Unreleased

### 改
- **海嘯再密 +10%**：spawn 間隔乘數 0.9 → 0.81（在前一輪 +10% 之上再加 10%，累積比原始快 ~23%）。
- **前期金幣加量 + 回復變快**（針對「前期太難」回饋）：
  - 金幣總量公式 `14 + zone × 10` → `16 + zone × 10 + max(0, 4 − zone) × 4`（zone 0 從 14 → 32、zone 1 從 24 → 38、zone 2 從 34 → 44；zone 5 以後幾乎沒變）。
  - 金幣 respawn 60s → 30s。每分鐘看到地上錢比拿掉的還多，撿不完。

### 新
- **💬 公屏聊天記錄**：左下聊天 log 不再 8 秒淡出，改成持續顯示（上限 50 條、超過自動捲動），加上時間戳。
  - Server 保留最近 50 條，新玩家進場時透過 init 訊息收到最近 30 條，所以斷線重連也看得到剛剛在講什麼。
  - log 框加了滾動條，可以往上看歷史訊息。

### 新
- **🏏 球棒打人 + 力量升級**：每個角色右手都拿著一根棕色木棒。按 <kbd>F</kbd>（手機點右側「揍」按鈕）對前方 6 公尺內的玩家揮棒。
  - 揮棒動畫：肩上往後 → 全力前揮（總共 0.4 秒、smoothstep 緩動），看起來像棒球揮棒。
  - 新增「**力量**」升級項（base $35）：每級擊退距離 +20 magnitude，可以把人打進海嘯裡。
    - Lv 0 ≈ 27 公尺 / Lv 5 ≈ 60 公尺 / Lv 10 ≈ 93 公尺（衰減 5%/秒、爆發感強）。
  - Server 權威：射程 6（球棒範圍）、冷卻 2 秒、強度上限 Lv 20。
  - 被打中：一定會擊退、**50% 機率**額外暈眩 0.5 秒（不能輸入，但重力照走、海嘯照打、坑照掉）。
  - 新音效：揮棒 swoosh + 命中 thwack（低 sawtooth + 中 square + 雜訊 burst）。
  - 抓人丟出去是下一波要做的事。

### 新
- **角色動畫更生動**
  - 跑步腳手擺動速度跟著實際移動速度（衝刺、加速 buff、瘦/壯身材都會反映在擺動頻率）
  - 新增**長髮**髮型選項（造型店、創角畫面）
  - **長髮 / 尖刺髮型跑步時會往後飄**：3 條髮絲 / 3 根尖刺各自掛在頭頂、靜止時下垂、移動時整體仰角向後，配合輕微擺動
  - 多人玩家會看到對方的擺動 + 飄髮（透過 `moving` flag 同步）

### 修
- **選完角色又跳回選角色（死循環）**：title-continue 不管有沒有 active profile 都直接顯示「選擇角色」清單，所以登入後重整看 reload 又跳到選擇。改成：有 active profile 就直接進遊戲；想切換用標題畫面新加的「切換 / 新增角色」小按鈕。
- **遊戲完全進不去**（空白畫面 / 點開始無反應）：上一版 `renderLeaderboard()` 在啟動時被呼叫，但裡面讀的 `scoreboardData` 用 `let` 宣告還在 temporal dead zone → 整個 script 從那行起就掛了。拿掉啟動時的 render 呼叫（之後 init / scoreboard_update 訊息來時自然會 render）。
- **排行榜永遠顯示「載入中…」**：實際是 server 還沒重 deploy，init 訊息裡沒有 scoreboard 欄位，client 完全收不到資料而 HTML 預設文字停在 loading。改成預設「尚無紀錄」，server deploy 之後會立即被 init 的 scoreboard 取代。

### 新
- **🏆 排行榜**：右上角顯示「最遠紀錄」前 5 名，自己金色高亮。
  - Server 端持久化（PartyKit Durable Object 的 `room.storage`），即使所有人都離線了紀錄也會留著。
  - 玩家每跑 5m 就回報新的最遠距離給 server，同名以最高紀錄為準（top 20 保留）。
  - 進場時 init 訊息含初始 scoreboard，之後有變動 server 廣播 `scoreboard_update`。
  - 持久化每 5 秒最多寫一次（避免 CF storage hammer）。

## 2026-05-10

### 新（多人連線）
- **PartyKit server** — Phase 1A 位置中繼，所有玩家共用 `global` 房間。
- **Phase 1B 共享世界**：海嘯生成、金幣、大寶箱、暴風雨全部移到 server。客戶端純渲染、撿取走 race 機制。
- **死亡寶箱共享**：玩家死亡時 server 廣播 `chest_spawn`，所有玩家都能撿；浮現「OO 的遺留物」標籤。
- **死亡寶箱會吃負重 + 切割**：包包剩多少格就拿多少，剩下灑成地上散幣（任何人可撿、不重生）。
- **線上玩家名單**：右上清單，加入 / 離開 / 改名即時更新。
- **聊天系統**：<kbd>Enter</kbd> 開輸入框、再 <kbd>Enter</kbd> 送出。頭上冒對話泡泡 5 秒、左下對話 log 保留 6 條（8 秒淡出）。

### 新（多角色 / Profile）
- localStorage 多 profile 儲存（`etfm-profile-<名字>`），每角色獨立紀錄存款 / 升級 / 造型 / 房屋 / 材料。

### 新（建造系統 — 方案 A 模組化）
