# Changelog

## 2026-05-10

### 新（多人連線）
- **PartyKit server** — Phase 1A 位置中繼，所有玩家共用 `global` 房間。
- **Phase 1B 共享世界**：海嘯生成、金幣、大寶箱、暴風雨全部移到 server。客戶端純渲染、撿取走 race 機制。
- **死亡寶箱共享**：玩家死亡時 server 廣播 `chest_spawn`，所有玩家都能撿；浮現「OO 的遺留物」標籤。
- **死亡寶箱會吃負重 + 切割**：包包剩多少格就拿多少，剩下灑成地上散幣（任何人可撿、不重生）。
- **線上玩家名單**：右上清單，加入 / 離開 / 改名即時更新。
- **聊天系統**：<kbd>Enter</kbd> 開輸入框、再 <kbd>Enter</kbd> 送出。頭上冒對話泡泡 5 秒、左下對話 log 保留 6 條（8 秒淡出）。

### 新（多角色 / Profile）
- localStorage 多 profile 儲存（`etfm-profile-<名字>`），每角色獨立紀錄存款 / 升級 / 造型 / 房屋 / 材料。
- 標題畫面 → 「選擇角色」清單，可進入 / 刪除 / 新建。
- 第一次升級舊存檔自動遷移到一個 profile。
- $50 整套造型重選只能改造型不能改名（鎖 input）。
- 「⚠ 重置進度」只清當前角色。

### 新（建造系統 — 方案 A 模組化）
- **後院腹地開放**：基地後方 120 × 70 綠地，9 個地塊（3 × 3，12 × 12 / 塊）。
- **買地塊**：<kbd>P</kbd>，每塊 $500。買下顯示施工告示牌 + 玩家名字。
- **建造選單**：<kbd>H</kbd>，5 槽位（地板 / 牆 / 屋頂 / 家具 / 裝飾）共 17 件物品（含玻璃牆 / 玻璃圓頂）。
- 每件物品消耗金錢 + 材料（木 / 石 / 鐵）。多人共享。

### 新（造型店）
- 按 <kbd>C</kbd> 開造型店：20 件搞怪造型分四類（帽 / 臉 / 背 / 配件）。
- 沒衣櫃時買新自動賣舊（退 1/3 價）。
- $50 整套造型重選按鈕。

### 新（道具系統）
- **加速 ⚡ / 跳高 🦘 / 磁鐵 🧲** 三種 buff 道具，地上發光柱、撿到觸發效果。
- Server 隨機 spawn 三種類型，45 秒重生（重生重新洗類型）。
- HUD 三種 buff 各自顯示倒數秒數。

### 新（大寶箱）
- z = −1000 放一個金色大寶箱（$50 直接入帳，10 分鐘重生，全圖最多 1 個）。

### 新（材料系統）
- 撿金幣有 7% 機率掉一個材料，淺區偏木頭、中區偏石頭、深區偏鐵。
- 掉落會在金幣位置顯示對應 3D 模型（圓木 / 多面體 / 錠塊），向上飄起淡出。
- 頂部資源欄顯示 $ / 🪵 / 🪨 / ⚙️。

### 新（玩家體驗）
- **跑道延長到 2km / 10 個區域**。
- **基地前的山坡拆除**，改成 100 × 20 木板大廣場（直接走過去就到）。
- **身材選擇**：瘦（速度 ×1.10）/ 標準 / 壯（×0.85），模型同步縮放。
- **金幣造型**：1 = 圓盤、2–9 = 疊堆、10+ = 棕色錢囊（球型 + $ 標）。
- **HUD 袋值欄**：除了「攜帶 5/10」也顯示金額 $25，避免誤以為攜帶 = 金額。
- **BGM 多檔輪播**：`assets/bgm/01.mp3 … 12.mp3` 順序循環。
- **音效 / BGM 獨立開關**：HUD 兩顆按鈕。
- **鏡頭縮放**：滾輪 / 兩指捏合（4 ~ 25 單位）；<kbd>R</kbd> 復位。
- **鏡頭旋轉**：滑鼠**左鍵**拖曳（避開右鍵選單）；觸控空白處拖曳。
- **15 種音效合成**（跳、撿錢、進坑、警報、暴風雨等）+ 4 個重點事件支援檔案覆蓋。
- **重置進度按鈕**：紅色雙確認。

### 改（玩法調整）
- **海嘯密度 +10%**（spawn 間隔 ×0.9）。
- **任何海嘯都可以跳過去**（只要跳得夠高）：
  - 矮浪 h=8 → Lv 3 / 標準 + 快 + 寬 h=15 → Lv 6 / 巨浪 h=25 → Lv 9 / 災難 h=50 → Lv 15
- **海嘯警報文字精簡**：只顯示浪名 + 顏色，不再強制顯示 Lv 需求。
- **坑系統重做**：拿掉 3 秒計時器。坑變成真實洞，按 <kbd>Space</kbd> 跳出來。跳高不夠的坑當實體牆走不過去。
- **「潛水時間」升級項目移除**，已購買的等級在下次載入自動退費。
- **撿死亡寶箱不再直接入帳**，改成佔背包格、要帶回廣場才存款。
- **道具放大 + 加色柱**（之前 0.9 立方體太小看不到）。

### 修
- 鏡頭 + 道具 + 海嘯一連串「隱形」bug 修復（材質 / 旋轉 / scene.add 漏掉 / camera far 不夠）。
- 坑爬出來瞬間又被吸回去（紀錄上次離開坑、走出範圍才能重進）。
- 平板搖桿不出現（懶偵測 + <kbd>T</kbd> 鍵手動切換）。
- 手機搖桿卡住（move / up 改綁 window + touchend 備援）。
- 音效靜音按鈕點不到（HUD pointer-events: none 擋住內部按鈕）。
- 進度沒存（localStorage 持久化）。
- 客戶端會偵測 server 是否為 Phase 1B，舊 server 自動 fallback 本地生成。

### 新（角色創建 + 教學 + 故事）

## 2026-05-06

### 修

- 第 1 個坑從 −22m 推到 −50m，給開場暖身。
- 第 1 區坑視覺縮小成 65%（給新手）。

### 新

- 跑道**寬度 20 → 40 人身**。
