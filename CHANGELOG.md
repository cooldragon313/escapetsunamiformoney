# Changelog

## Unreleased

### 新
- **重置進度按鈕**：角色創建畫面底部紅色虛線「⚠ 重置所有進度」連結，按下後彈出第一個紅色確認窗 → 再彈第二個深紅色「最後確認」窗 → 才會清掉 localStorage 並重新載入。雙保險防誤觸。

### 修
- **音效靜音按鈕點不到**：HUD 整層設了 `pointer-events: none` 讓玩家可以點背後的遊戲畫面，連帶擋住了內部按鈕的點擊。在 `#audio-toggle` 上加 `pointer-events: auto` 例外讓它可以收到點擊。

### 新
- **BGM 多檔輪播**：建立 `assets/bgm/` 資料夾。把 mp3 命名 `01.mp3, 02.mp3 … 12.mp3` 放進去，會自動依編號順序播放並循環，每首結束接下一首。原本的 `assets/bgm.mp3` 仍當第一首播放。
- **音效 / BGM 獨立開關**：HUD 改成兩個按鈕 — 🔊 音效 / 🎵 音樂，各自可切換、各自存 localStorage（`etfm-sfx` / `etfm-bgm`）。舊偏好 `etfm-audio` 自動遷移到兩邊。
- **加速道具**（紅色發光方塊）：跑道上每區 1–2 顆，撿到 +50% 跑速 10 秒（可重複撿延長）。同金幣機制 — server 權威，先到先贏，45 秒重生。HUD 加一行顯示剩餘秒數。
- **跳高道具 🦘**（綠色）+ **磁鐵道具 🧲**（藍色）— 跳高 +50% 5 秒；磁鐵把撿取範圍 ×2.5 5 秒。Server 端隨機 spawn 三種類型，每次重生重新洗牌類型。HUD 三種 buff 各自顯示倒數。
- **死亡寶箱共享**：玩家死亡時 server 廣播 `chest_spawn`，所有玩家都看得到，誰都能撿。
- **線上玩家名單**：右上角顯示目前同房間的所有玩家名字，加入 / 離開 / 改名即時更新。
- **HUD 顯示袋值**：「攜帶 5 / 10」之外多一行「袋值 $25」，避免誤以為「攜帶 = 金額」。深區金幣 / 暴風雨倍率反映在袋值。
- **道具放大 + 加色柱**：原本 0.9 立方體太小不顯眼，改成 1.4 立方體 + 6 高色柱 + 白頂帽。三色（紅 / 綠 / 藍）遠遠就看得到。
