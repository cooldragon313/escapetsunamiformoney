# BGM 資料夾

把 MP3 檔放這裡會自動加入循環播放清單。

## 命名規則

要被載入的檔案必須命名為**兩位數編號**：

```
01.mp3
02.mp3
03.mp3
...
12.mp3
```

最多支援 12 首。其他檔名會被忽略。

## 播放順序

- 按編號順序播放：01 → 02 → 03 → … → 12 → 01（循環）
- 每首結束後自動接下一首
- 整個清單跑完一輪會從第一首重來

## 既有的 `assets/bgm.mp3`

如果 `assets/bgm.mp3`（單檔）還在，會被排在所有編號檔的**前面**當第一首。
你可以：

- 留著 `assets/bgm.mp3` + 在這資料夾放 `01.mp3 02.mp3 …` → 順序：bgm.mp3 → 01 → 02 → …
- 把 `assets/bgm.mp3` 改名 / 移進這個資料夾當 `01.mp3` → 純編號管理
- 完全不放編號檔 → 只播 `assets/bgm.mp3`

## 建議

- 大小：每首 < 5 MB（GitHub Pages 載入要時間）
- 格式：MP3 / 128 kbps 通常夠
- 風格：Loop 友善的曲子（結尾接得回頭部最自然）
