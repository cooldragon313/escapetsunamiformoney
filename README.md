# Escape Tsunami For Money

第三人稱 voxel 跑酷網頁遊戲，靈感來自 Roblox《Escape Tsunami For Brainrots!》。
撿金幣、躲海嘯、回基地存款 / 升級 / 蓋房子 / 換造型，再衝更深。

**直接玩**：https://cooldragon313.github.io/escapetsunamiformoney/

純前端，瀏覽器打開就玩、手機 / 平板自動切觸控介面。多人世界，最多 100 人共用一個跑道。

## 玩法循環

1. 從廣場出發 → 跑到危險區撿金幣（深區面額更大）
2. 看到海嘯警報 → 跳過去（跳得夠高）/ 跳坑躲 / 撤退
3. 撿到 7% 機率掉**建材**（木 / 石 / 鐵）
4. 活著回廣場 → 自動把背包換成存款
5. 在廣場按 <kbd>B</kbd> 升級裝備、按 <kbd>C</kbd> 買造型
6. 跑到後院（廣場後方）→ <kbd>P</kbd> 買地塊（$500）→ <kbd>H</kbd> 蓋房子
7. 死了：背包變成「OO 的遺留物」寶箱，所有玩家都能撿

## 海嘯

10 個難度區（每 200m 一區）。6 種海嘯共享同一個世界 — 大家看到同一條浪。

| 名稱 | 顏色 | 高 | 寬 | 速度 | 跳得過去需 跳高 Lv | 出現率 |
|---|---|---|---|---|---|---|
| 矮浪 | 綠 | 8 | 14 | × 1.0 | 3 | 25% |
| 標準 | 藍 | 10 | 18 | × 1.0 | 4 | 34% |
| 快浪 | 紅 | 10 | 14 | × 1.8 | 4 | 20% |
| 寬浪 | 深藍 | 10 | 32 | × 0.85 | 4 | 16% |
| 巨浪 | 紫 | 20 | 全寬 | × 0.7 | 8 | 4% |
| 災難 | 黑 | 50 | 全寬 | × 0.5 | **15（基本不可能）** | 1% |

**95% 以上的浪只要 Lv 4 跳高就跳得過去**。災難浪（黑）需要 Lv 15，幾乎是必躲坑或撤退的等級。
部分寬度的浪會**左右擺動**，可以朝它走離的方向閃。

### 暴風雨

偶發事件（每 30–80 秒可能觸發、持續 14–22 秒）：
- 海嘯密集連發（0.8–2.4 秒一波）
- 撿到的金幣價值 ×2
- 巨浪 / 災難浪出現機率提高
- 天空轉紅 + HUD 顯示 ⚡ 暴風雨 ⚡

## 坑（避難所）

坑現在是**真正的洞**，重力會把你帶下去。**按 <kbd>Space</kbd> 跳出來**，跳得夠高就出來。

跳高 Lv 不夠的坑會被當作**實體牆**，連走過去都不行。所以深區（z=−1000+）沒練跳高就是鐵躺。

| 區域 | 距離 | 坑緣顏色 | 需要跳高 Lv |
|---|---|---|---|
| 1 | 0–200m | 棕 | 0 |
| 2 | 200–400m | 綠 | 1 |
| 3 | 400–600m | 藍 | 2 |
| 4 | 600–800m | 紫 | 3 |
| 5 | 800–1000m | 紅 | 4 |
| 6–10 | 1000–2000m | 橙 / 金 / 青 / 桃 / 黑 | 5–9 |

## 廣場（安全區）

紅色安全線後是 100 × 20 的木板大廣場 — 海嘯到此停下。

- **走到廣場就自動兌換背包成存款**

- 隨時可按 <kbd>B</kbd> 開升級店、<kbd>C</kbd> 開造型店

| 屬性 | 效果 |
|---|---|
| 跑速 | +0.04 / 級 |
| 跳高 | +0.08 / 級（解鎖深區坑、讓你跳過更高的浪） |
| 撿取範圍 | +0.4 / 級 |
| 負重 | +5 / 級 |
| 力量 | 揮棒擊退距離 +20 magnitude / 級（Lv 0 ≈ 27m / Lv 10 ≈ 93m）|
| ~~潛水時間~~ | 已移除（坑不再有計時器，舊存檔自動退費） |

## 後院 / 蓋房子（方案 A 模組化）

廣場後方（z = 22 ~ 78）120 × 70 綠地，9 個地塊以 3×3 排列。

- **買地塊**：<kbd>P</kbd>，每塊 $500。買下中央會有施工告示牌
- **建造**：站在自己的地塊上 <kbd>H</kbd> 開選單

每塊地有 5 個槽位、共 17 件可建造物：

| 槽 | 選項 |
|---|---|
| 地板 | 木板 / 石磚 / 沙地 |
| 牆 | 木牆 / 石牆 / 鐵牆 / **玻璃牆** |
| 屋頂 | 平頂 / 紅瓦斜頂 / 藍色斜頂 / **玻璃圓頂** |
| 家具 | 床 / 桌椅 / 寶箱 |
| 裝飾 | 藍旗 / 盆栽 / 路燈 |

每槽只能裝一件，買新的覆蓋舊的（沒退費）。多人共享 — 別人看得到你蓋的房。

## 多人連線

伺服器：[PartyKit](https://www.partykit.io/) Durable Object。所有玩家共用 `global` 房間：

- **同一條海嘯**：server 為權威，spawn / 速度 / 位置同步
- **金幣搶先撿**：race 機制，誰先到誰拿
- **共享暴風雨 / 大寶箱 / 道具 / 死亡寶箱 / 地塊**
- 每個玩家頭上飄名字標籤
- 右上「線上玩家」清單

### 聊天

按 <kbd>Enter</kbd> 開輸入框，再按 <kbd>Enter</kbd> 送出（<kbd>Esc</kbd> 取消）：

- **頭上冒對話泡泡**（5 秒）— 所有玩家都看到
- **左下對話 log**（最近 6 條，8 秒淡出）
- 訊息上限 80 字

## 帳號系統（跨裝置）

雲端帳號：用同一個帳密在任何裝置登入，存款 / 升級 / 造型 / 房屋 / 排行榜紀錄全部同步。

- 第一次玩 → 標題 → 註冊（帳號 3-16 個英數字底線，顯示名稱 1-16 字、之後不能改）
- 已有帳號 → 標題 → 登入
- 標題畫面下方「登出 / 切換帳號」清掉本機 token 後重新登入
- 排行榜以帳號 ID 為主、顯示名稱備註，**他人無法冒用你的名字刷分**
- 「⚠ 重置進度」會清掉伺服器上這個帳號的所有資料（帳號本身保留）

存檔位置：伺服器端的 PartyKit Durable Object storage，key 為 `user:<lowercase username>`。localStorage 只當作離線快取，**不再是真實來源**。

> 沒有忘記密碼流程，請用記得住的密碼。

## 部署 PartyKit Server

如果你要 fork 自己玩，server 要重新部署：

```bash
npm install
npx partykit login                              # 第一次：GitHub OAuth

# 設定 auth 的 HMAC 簽章金鑰（不要外流；換 key 會讓全部 token 失效）
npx partykit env add AUTH_SECRET                # 互動輸入隨機字串
# 或一次塞：
# echo "$(openssl rand -hex 32)" | npx partykit env add AUTH_SECRET

npm run deploy                                  # 印出網址
```

把網址（不含 `https://`）貼到 `index.html`：

```js
const PARTYKIT_HOST = 'etfm.<你的帳號>.partykit.dev';
```

然後 `git push` 到 GitHub Pages。**留空 `PARTYKIT_HOST` 不再退回單人模式**——帳號系統需要伺服器，沒設好的話遊戲開不起來。

### 本機開發

```bash
# 本機環境一樣需要 secret；用 .env 檔
echo 'AUTH_SECRET=dev-only-replace-this' > .env
npm run dev    # localhost:1999
```

把 `PARTYKIT_HOST = 'localhost:1999'` 設好，瀏覽器打開 `index.html` 即可。
