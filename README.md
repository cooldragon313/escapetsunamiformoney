# Escape Tsunami For Money

第三人稱 voxel 跑酷網頁遊戲，靈感來自 Roblox《Escape Tsunami For Brainrots!》。
撿金幣、躲海嘯、回基地存款、升級、再衝更深。

**直接玩**：https://cooldragon313.github.io/escapetsunamiformoney/

純前端，瀏覽器打開就玩，不用安裝任何東西。手機 / 平板自動切到觸控介面。

---

## 故事背景

> 傳說大海嘯之國的盡頭，
> 沉著一筆無人能取的寶藏。
>
> 無數英雄前來，沒有一個從第五區活著回來。
> 山頂上的老人說：「那是給最英勇者的禮物。」
>
> 你能跑到 1000 公尺嗎？

## 操作

| 操作 | 桌面 | 手機 / 平板 |
|---|---|---|
| 移動 | <kbd>WASD</kbd> / 方向鍵 | 左下虛擬搖桿 |
| 跳躍 | <kbd>Space</kbd> | 右下「跳」按鈕 |
| 衝刺 | <kbd>Shift</kbd>（按住） | 右側「衝刺」按鈕 |
| 開商店 | <kbd>B</kbd>（限山頂） | 右上「商店」按鈕 |

---

## 玩法循環

1. 從山頂基地出發，往前（−Z 方向）跑進危險區
2. 沿途撿金幣，深區金幣更值錢
3. 看到海嘯警報 → 躲坑或跑回安全區
4. 活著爬上山頂 → 自動把背包換成存款
5. 在山頂按 B 升級 5 種屬性：跑速 / 跳高 / 潛水時間 / 撿取範圍 / 負重
6. 死了：寶藏掉在死亡處變大金箱，下一條命可以回去撿

## 海嘯類型

| 名稱 | 顏色 | 高 | 寬 | 速度 | 處理方式 |
|---|---|---|---|---|---|
| 矮浪 | 綠 | 3 | 14 | 1.0× | 跳起來就過 |
| 標準 | 藍 | 6 | 18 | 1.0× | 躲坑或側閃（會擺動） |
| 快浪 | 紅 | 7 | 14 | 1.8× | 反應要快、躲坑或側閃 |
| 寬浪 | 深藍 | 6 | 32 | 0.85× | 較難閃避，多半要躲坑 |
| 巨浪 | 紫 | 14 | 全寬 | 0.7× | 必躲坑（暴風雨多） |
| 災難 | 黑 | 22 | 全寬 | 0.5× | 必躲坑（暴風雨多） |

部分寬度的海嘯會邊推進邊**左右擺動**，可以朝它走離的方向閃。

## 暴風雨

偶發事件（每 30–80 秒可能觸發一次，持續 14–22 秒）：
- 海嘯密集連發（0.8–2.4 秒一波）
- 撿到的金幣價值 ×2
- 巨浪／災難浪出現機率提高
- 天空轉紅，HUD 顯示 ⚡ 暴風雨 ⚡

## 坑（避難所）

坑會用顏色標示需要的**跳高 Lv**：

| 區域 | 距離 | 坑緣顏色 | 需要跳高 Lv |
|---|---|---|---|
| 1 | 0–200m | 棕 | 0（任何人）|
| 2 | 200–400m | 綠 | 1 |
| 3 | 400–600m | 藍 | 2 |
| 4 | 600–800m | 紫 | 3 |
| 5 | 800–1000m | 紅 | 4 |

跳高不夠走過去 → 顯示提示，**不會把你關進去**，繼續跑就好。
跳得進去的坑會固定保護你 3 秒，然後自動爬出。

---

## 山坡 / 安全區

過了紅色安全線後是 7 階斜坡：

- 海嘯到山腳會消失，山坡上完全免疫
- 坡頂有金色光柱、商店招牌、旗子
- 走到頂階 → 自動把背包換成存款 + HUD 提示按 B 開商店

## 多人連線（PartyKit）

伺服器使用 [PartyKit](https://www.partykit.io/)（Cloudflare 邊緣節點）。
每個房間是一個 Durable Object，玩家之間透過 WebSocket 中繼位置。

### 部署步驟（一次性）

```bash
# 1. 安裝依賴
npm install

# 2. 第一次登入（會開瀏覽器走 GitHub OAuth）
npx partykit login

# 3. 部署 server，會印出像 https://etfm.<你的帳號>.partykit.dev
npm run deploy
```

把 deploy 後的網址（不含 `https://`）貼到 `index.html` 開頭：

```js
const PARTYKIT_HOST = 'etfm.<你的帳號>.partykit.dev';
```

然後 `git push` 到 GitHub Pages。所有玩家會共用同一個房間 (`global`)。

### 本機開發

```bash
npm run dev
```

會在 `localhost:1999` 啟動。把 `PARTYKIT_HOST = 'localhost:1999'` 設好，
直接打開 `index.html`（瀏覽器） 就能測試多人。
